h1. Initial Setup

* Clone the git repository to a local directory, and clone the Textile Editor Helper.  To do this, execute the following four command in a terminal

<pre>
	git clone git://github.com/jcxplorer/hhlc.git

	cd hhlc

	git submodule init

	git submodule update
</pre>

* Create the config/database.yml

E.g.
<pre>
development:
    adapter: sqlite3
    database: db/development.sqlite3
    pool: 5
    timeout: 5000
</pre>

* Run 'rake setup' in the local directory

* Run 'rake db:populate' to add test data to the database (optional)
